This is a Beta version of a metalation calculator that will later become part of the E3B Metals in Biology site. Please note that it is currently in development so changes and bugs should be expected.

If you spot any issues with the calculator, please log an issue in GitHub or contact: metals.bbsrcnibb@durham.ac.uk.

Metalation Calculator

Predict metal occupancies in vivo

To predict the metalation state of a protein or molecule, fill in values in the table for as many determined metal affinities (and availabilities if known) as possible.
